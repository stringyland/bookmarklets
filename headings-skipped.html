This is a test case designed to determine how the Easy Checks heading bookmarklet deals with a heading structure that skips levels.

HTML example

Start a new heading "section" of the page with H1

But skip past H2

Then carry on with correct levels

ARIA example

Start a new heading "section" of the page with an ARIA heading
But skip past level 2
Then carry on with correct levels

Results

Expected

  • The second heading in both example sections should be flagged as an error.
  • The error flag should describe the problem.
  • Other headings from the page are displayed with no warnings or errors.

Actual as of 2023-09-05

  • The second headings are correctly flagged with an error style.
  • The error style does not include a description of the problem.
