Results

Expected

  • The visually-hidden headings should be displayed in the panel's list with no warnings or errors.
  • The hidden-all headings should create a skipped level error on the following heading.
  • The aria-hidden headings should create a skipped level error on the following heading.
  • The Show Hidden toggle lists the number of hidden headings, and toggles their visibility in the panel.
  • The Mark Hidden toggle creates a distinct visual style for visually-hidden headings.
  • Other headings from the page are displayed with no warnings or errors.
