This is a test case designed to determine how the Easy Checks heading bookmarklet deals with incorrectly marked up ARIA headings.

Results

Expected

  • For each heading in the example section, it is either not listed in the panel, or is listed with an error flagged.
  • Each error flag should describe the problem.
  • Other headings from the page content displayed with no warnings or errors.

Actual as of 2023-09-05

  • Missing aria-level: flagged by NaN (not a number). No style change.
  • The remaining example headings are not listed in the panel.
