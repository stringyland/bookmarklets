This test case is designed to determine how role="region" is announced by various screen readers.

By default, the element being tested does not receive focus, so the only way to test how it is announced is via:

  • VoiceOver: Announce next item: VO + Right Arrow
  • NVDA and JAWS: Announce next line: DOWN ARROW

Example

Test content

Code

<div role="region">
  <h4>Test content</h4>
</div>

Assistive technologies

VoiceOver

  • Chrome: Heading level 4. Test content. [Region role not announced]
  • Firefox: Heading level 4. Test content. [Region role not announced]
  • Safari: Heading level 4. Test content. [Region role not announced]

NVDA

  • Chrome: Heading level 4. Test content. [Region role not announced]
  • Firefox: Heading level 4. Test content. [Region role not announced]
  • Edge: Heading level 4. Test content. [Region role not announced]

JAWS

  • Chrome: Heading level 4. Test content. [Region role not announced]
  • Firefox: Heading level 4. Test content. [Region role not announced]
  • Edge: Heading level 4. Test content. [Region role not announced]
