This is a test case designed to determine how the <input type="tel"> element is announced by various screen readers.

Example

Code

<div>
  <label for="tel">Enter phone number</label>
  <input type="tel" id="tel">
</div>

Assistive technologies

VoiceOver

  • Chrome: Enter phone number. Edit text.
  • Firefox: Enter phone number. Edit text.
  • Safari: Enter phone number. Edit text.

NVDA

  • Chrome: Enter phone number. Edit. Blank.
  • Firefox: Enter phone number. Edit. Has autocomplete. Blank.
  • Edge: Enter phone number. Edit. Blank.

JAWS

  • Chrome: Enter phone number. Edit. Type in text.
  • Firefox: Enter phone number. Edit. Type in text.
  • Edge: Enter phone number. Edit. Type in text.
