This is a test case designed to determine how the <input type="checkbox"> element is announced by various screen readers.

Example

Code

<div>
  <input type="checkbox" id="subscribe">
  <label for="subscribe">Subscribe</label>
</div>

Assistive technologies

VoiceOver

  • Chrome: Subscribe. Unticked. Tick box.
  • Firefox: Subscribe. Unticked. Checkbox.
  • Safari: Subscribe. Unticked. Tick box.

NVDA

  • Chrome: Subscribe. Checkbox. Not checked.
  • Firefox: Subscribe. Checkbox. Not checked.
  • Edge: Subscribe. Checkbox. Not checked.

JAWS

  • Chrome: Subscribe. Checkbox. Not checked.
  • Firefox: Subscribe. Checkbox. Not checked.
  • Edge: Subscribe. Checkbox. Not checked.
