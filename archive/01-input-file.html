This is a test case designed to determine how the <input type="file"> element is announced by various screen readers.

Example

Code

<div>
  <label for="file">Choose a file</label>
  <input type="file" id="file">
</div>

Assistive technologies

VoiceOver

  • Chrome: Choose a file. Button.
  • Firefox: Browse.... Button. Choose a file. Group.
  • Safari: No file selected. Choose a file. File upload button.

NVDA

  • Chrome: Choose a file. Button.
  • Firefox: Choose a file. Grouping. Browse. Button.
  • Edge: Choose a file. Button.

JAWS

  • Chrome: Choose a file. Button.
  • Firefox: Choose a file. Browse. Button.
  • Edge: Choose a file. Button.
