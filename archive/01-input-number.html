This is a test case designed to determine how the <input type="number"> element is announced by various screen readers.

Example

Code

<div>
  <label for="number">Choose a number</label>
  <input type="number" id="number">
</div>

Assistive technologies

VoiceOver

  • Chrome: Choose a number. Stepper.
  • Firefox: Choose a number. Stepper.
  • Safari: Choose a number. Incremental. Edit text.

NVDA

  • Chrome: Choose a number. Spin button. Editable. Blank.
  • Firefox: Choose a number. Spin button. Has autocomplete. Editable. Blank.
  • Edge: Choose a number. Spin button. Editable. Blank.

JAWS

  • Chrome: Choose a number. Edit. Spin box.
  • Firefox: Choose a number. Edit. Spin box.
  • Edge: Choose a number. Edit. Spin box.
