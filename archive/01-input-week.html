This is a test case designed to determine how the <input type="week"> element is announced by various screen readers.

Example

Code

<div>
  <label for="week">Choose a week</label>
  <input type="week" id="week">
</div>

Assistive technologies

VoiceOver

  • Chrome: Week. Choose a week. Stepper.
  • Firefox: Choose a week. Edit text.
  • Safari: Choose a week. Edit text.

NVDA

  • Chrome: Choose a week. Date edit. Week. Choose a week. Spin button. 0.
  • Firefox: Choose a week. Edit. Has autocomplete. Blank.
  • Edge: Choose a week. Date edit. Week. Choose a week. Spin button. 0.

JAWS

  • Chrome: Choose a week. Date time edit. Week. Choose a week. Spin button. 0.
  • Firefox: Choose a week. Edit. Type in text.
  • Edge: Choose a week. Date time edit. Week. Choose a week. Spin button. 0.
