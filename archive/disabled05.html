<option> with disabled

← Return to main list

This is a test case designed to determine how the disabled attribute, applied to the <option> element, is announced by various screen readers.

Disabled attributes cannot receive focus, so the only way to test how these attributes are announced is via:

  • VoiceOver: Announce next item: VO + Right Arrow
  • NVDA and JAWS: Announce next line: DOWN ARROW

Example

Code

<div>
  <label for="suburb">Suburb</label>
  <select id="suburb">
    <option value="Abbotsford" disabled>Abbotsford</option>
  </select>
</div>

Assistive technologies

VoiceOver

  • Chrome: Abbotsford. Dimmed.
  • Firefox: [Option not announced]
  • Safari: Abbotsford. Dimmed.

NVDA

  • Firefox: [Option not announced]
  • Edge: [Option not announced]
