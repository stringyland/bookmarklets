This is a test case designed to determine how the <input type="url"> element is announced by various screen readers.

Example

Code

<div>
  <label for="url">Enter web address</label>
  <input type="url" id="url">
</div>

Assistive technologies

VoiceOver

  • Chrome: Enter web address. Edit text.
  • Firefox: Enter web address. Edit text.
  • Safari: Enter web address. Edit text.

NVDA

  • Chrome: Enter web address. Edit. Blank.
  • Firefox: Enter web address. Edit. Has autocomplete. Blank.
  • Edge: Enter web address. Edit. Blank.

JAWS

  • Chrome: Enter web address. Edit. Type in text.
  • Firefox: Enter web address. Edit. Type in text.
  • Edge: Enter web address. Edit. Type in text.
